# 需求
## 关于表格高度
在业务中，我们的表格高度一般设置为可用空间的100%，或者根据数据的数量定高。第一种情况，高度是需要选然后计算的，计算后得到一个固定的值。后者，则根据行内容的实际渲染高度决定。所以，需要一个开关来决定是否使用虚拟渲染比较合理。或者可以根据高度参数是否是一个固定的值来推断是否需要可视区域渲染。

参数起名-是否可视区域渲染 - isVisibleAreaRender
